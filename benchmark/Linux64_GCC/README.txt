README.TXT
==============

This benchmark shows the time spent by each algorithm in the sorting
For to compile  and run the programs execute the shell script 

./compile_and_run.sh

The compilation need around 3 , 4 minutes.
The execution need between 5 to 20 minutes depending of your processor.
When finish, the results are in the file Results.txt

If you want see the benchmarks of this library with other non free SW, you can 
find mode detailed benchmarks, the code and procedures for to execute, in

https://github.com/fjtapia/sort_parallel_benchmark
