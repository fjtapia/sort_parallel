Detailed Description

template<uint32_t Block_size, uint32_t Group_size, class Iter_t, class Compare = util::compare_iter< Iter_t >>
struct boost::sort::parallel::detail::block_indirect_sort< Block_size, Group_size, Iter_t, Compare >

This class is the entry point of the block indirect sort. The code of this algorithm is divided in several classes: bis/block.hpp : basic structures used in the algorithm bis/backbone.hpp : data used by all the classes bis/merge_blocks.hpp : merge the internal blocks bis/move_blocks.hpp : move the blocks, and obtain all the elements phisicaly sorted bis/parallel_sort.hpp : make the parallel sort of each part in the initial division of the data.
