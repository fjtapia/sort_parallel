constructor of the class

Parameters
first: iterator to the first element of the range to sort
last: iterator after the last element to the range to sort
cmp: object for to compare two elements pointed by Iter_t iterators
num_thread: Number of threads to use in the process. When this value is lower than 2, the sorting is done with 1 thread
paux: pointer to the auxiliary memory. If nullptr, the memory is created inside the class
naux: number of elements of the memory pointed by paux
